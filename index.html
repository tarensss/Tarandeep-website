    PM PORTAL


      Project Overview: The PM Portal was developed to address inefficiencies in project management within our organization. By leveraging AWS technologies, we aimed to create an internal portal that would enable better tracking of project timelines, budgets, and resource allocations, thereby improving team collaboration and efficiency.


      Technologies Used: AWS Lambda, AWS DynamoDB, AWS S3, AWS CloudFormation, Node.js, and React.


      Methodology: We adopted an Agile development process, allowing for iterative updates based on user feedback. This approach ensured the portal met the evolving needs of project managers and team members.


      Challenges: One significant challenge was ensuring the scalability and security of the portal, given its extensive use across the organization. Implementing AWS CloudFormation allowed us to manage resources efficiently, while AWS IAM roles provided the necessary security measures.

      Outcomes: The portal significantly improved project visibility, allowing for real-time updates on project status, budget tracking, and resource allocation. It fostered a collaborative environment and increased the overall productivity of project teams.

    DASHBOARD APP



      Project Overview: The Dashboard App was conceived as a tool to monitor and manage various metrics and KPIs in real time. Developed using the MERN stack, it included features like user authentication, secure data handling, and dynamic data visualization.

      Technologies Used: MongoDB, Express.js, React, Node.js, JWT for authentication, and various data visualization libraries.
